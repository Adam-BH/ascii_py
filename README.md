# Ascii_py

A python package for making ascii art

## Installation

Run the following to install:

```python
$ pip install ascii_py
```

## Usage

### Terminal

```python
$
```

### Python script

```python
import ascii_py

```

# Developing Ascii_py

To install ascii_py, along with the tools you need to develop and run tests, run the following in your virtualenv:

```python
$ pip install -e .[dev]
```
